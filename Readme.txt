# MADIS Readme
MADIS- A Mask Detection and Identity Identification System

# Download MADIS

Step 1. :Download MADIS from (https://github.com/ravindranath-sawane/MADIS_Mask-Detection-and-Identity-Identification-System) Latest Version.
Step 2. :Extract all files into your C: Drive.

# Installation

using pip install install all required libraries mentioned below
tensorflow>=1.15.2
keras==2.3.1
imutils==0.5.3
numpy==1.18.2
opencv-python==4.2.0.*
matplotlib==3.2.1
scipy==1.4.1
pytesseract==v0.3.7

Download 3rd Party API Tesseract

tesseract-ocr from (https://github.com/UB-Mannheim/tesseract/wiki)

# Training Model
Mask's Trained Model is Already available if you want to train on your PC you can using (Mask_Training_Model.py)
The Model will be Saved as (mask_detector.model) at the same directory.

# MADIS Main
You can See a main folder in the Directory in that there is file called (Mask_ID_Detection.py)
You can run that file directly.

The program will start.
